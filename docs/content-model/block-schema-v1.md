# Block Schema V1

The first page editor version will use module-based drag and drop, not a pixel-level free canvas.

Planned block types:

- `hero`
- `dark_visual`
- `text_image`
- `image_collage`
- `case_list`
- `article_list`
- `solution_list`
- `faq`
- `cta`
- `video`
- `process`
- `capability`

Every public block should support the GEO basics it needs: readable headings, body HTML, internal links, media references, and image alt text.
