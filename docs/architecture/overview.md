# NEED Website Architecture Overview

The final NEED website system will use:

- Existing React + Vite frontend visual components.
- A graphical admin interface for non-developers.
- A Node.js + Express + TypeScript API service.
- MySQL 8 in production.
- Local uploads during development and Tencent Cloud COS in production.
- Static HTML generation during publishing for GEO and search visibility.

This round only creates the project skeleton. It does not implement database access, uploads, Word import, page editing, or static publishing.
